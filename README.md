# Project Register

Projeto de estudo, usando TypeScript, Nodejs e Nestjs.

## Inicialização

    git clone https://github.com/lucasbentu/project-register
    
    cd project-register
    
    npm install

    npm run start

Rotas: 

    GET:            /register
    GET:ID          /register/ID
    PUT:ID          /register/ID
    POST            /register
    DELETE:ID       /register/ID
